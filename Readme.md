Project scripts and usage

1. Number guessing game — interactive script
   - File: [student-management-system.py](student-management-system.py)
   - Run: `python student-management-system.py`

2. Simple calculator — REPL with history
   - File: [simple_calculator.py](simple_calculator.py)
   - Run: `python simple_calculator.py`

3. Password generator — secure password CLI
   - File: [password_generator.py](password_generator.py)
   - Run: `python password_generator.py`
   - Optional dependency: `pyperclip` (to copy password to clipboard)

4. Student Management System — simple file-backed CRUD
   - File: [student_management_system.py](student_management_system.py)
   - Run: `python student_management_system.py`

Dependencies
- To enable clipboard copying install:

```bash
pip install pyperclip
```

Notes
- All scripts are small command-line utilities designed for learning and quick use.
